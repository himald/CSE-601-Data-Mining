1. Open pca_script.py/svd_script.py/tSNE_script.py for PCA/SVD/t-SNE respectively and hard code the dataset name as an argument to the calling method.
Ex:
PCA: pca("pca_demo")
SVD: svd_impl("pca_demo")
t-SNE: tSNE_impl("pca_demo")

2. Run commands:	
python pca_script.py
python svd_script.py
python tSNE_script.py

3. Observe plots
