DBSCAN:

1. Run command:
python dbscan.py

2. Enter console input:
Enter input data: <full file name with extension>
Enter epsilon(radius) value: <radius value>
Enter the minimum number of points: <MinPts value>

Example:
Enter input file: cho.txt
Enter epsilon(radius) value: 1.1
Enter the minimum number of points: 3

3. Output:

Cluster: count gives the number of points in each cluster. -1 is noise.

Cluster: points in cluster gives each present present in its respective cluster.
